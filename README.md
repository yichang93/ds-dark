# DS-DARK (WIP)

DS-DARK is a dark mode design system that build based on [Ant Design](https://ant.design/). 


## Available Scripts

```
// Runs the app in the development mode.
$ yarn start

//Builds the app for production to the `build` folder.
$yarn build

```
